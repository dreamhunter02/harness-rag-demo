# Harness-1 Live Search Demo

Presentation demo comparing Harness-1 20B running behind a state-externalizing retrieval harness with a GPT-4o search baseline.

The local laptop hosts the UI, API, retrieval index, and harness state. A Brev GPU serves only the Harness-1 checkpoint through vLLM.

See [PLAN.md](PLAN.md) for the approved implementation plan. Setup and event-day instructions will be added as implementation milestones land.
